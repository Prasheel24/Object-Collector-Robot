35 ## Build
36 Steps to build
37 ```
38 mkdir -p ~/catkin_ws/src
39 cd ~/catkin_ws/
40 catkin_make
41 source devel/setup.bash
42 cd src/
43 git clone https://github.com/Prasheel24/Object-Collector-Robot
44 cd ~/catkin_ws/
45 catkin_make
46 ```
47 ## Run
48 **Generate Map**
49 <br> Open a new Terminal
50 ```
51 source devel/setup.bash
52 roslaunch ecobot TurtlebotMapping.launch
53 ```
54 **Save Map to Directory**
55 <br> Open a new Terminal
56 ```
57 rosrun map_server map_saver map:=/robot_1/map -f CollectorSpace
58 ```
59 **Collect Garbage**
60 <br> Open a new Terminal
61 ```
62 source devel/setup.bash
63 roslaunch ecobot TurtlebotCollect.launch
64 ```
